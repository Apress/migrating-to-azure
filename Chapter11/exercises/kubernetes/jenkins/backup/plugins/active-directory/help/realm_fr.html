Cette option permet à Jenkins d'authentifier un utilisateur avec mot de passe dans un domaine Active Directory.

Jenkins reconnaît tous les groupes Active Directory aux quels appartient l'utilisateur, vous pouvez donc vous en servir pour autoriser ou non l'accès aux ressources (par exemple vous pouvez choisir l'autorisation par matrice et autoriser les membres du groupe "Administrateurs du domaine" à administrer Jenkins.)
