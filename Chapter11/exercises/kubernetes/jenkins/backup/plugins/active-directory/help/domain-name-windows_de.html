(Optional) Spezifiziert das Active Directory, gegen das authentifiziert werden soll.

Ohne Angabe nutzt Jenkins ADSI um mit dem Active Directory Kontakt aufzunehmen. Dieses benötigt keinen konfigurierten Domänennamen.

Es wurden allerdings Probleme gemeldet, wenn Jenkins unter einem Benutzerkonto läuft, das nicht Mitglied der Domäne ist. Dies ist häufig dann der Fall, wenn Jenkins als Dienst installiert wird. In solchen Fällen, muss der Domänenname hier eingetragen werden. Jenkins nutzt dann LDAP für die Kommunikation mit dem Active Directory.
